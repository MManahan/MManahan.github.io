Summary: Latitude vs. X

The purpose of this project was to establish familiarity with making API calls using python in order to gather weather data based for specific cities given their latitude and longitude. The API used was OpenWeather API. Calls were made to gather max temperature, humidity, cloudiness and wind speed from 500+ cities. A pandas dataframe was then populated based on the data gathered and analyses were run.

Matplotlib was used to show the how proximity to the equator (based on a city's latitude) affects the max temperature, humidity, cloudiness and wind speed.

This dashboard provides the visualizations created to provide insight from the source data gathered.
